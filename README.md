## API Reference

<!-- automd:jsdocs src="src/commands/hello.ts" headingLevel=2 -->

### `hello(name)`

greet関数は、nameを受け取って"Hello, {name}!"というメッセージを出力します。

**Example:**

```js
greet("UnJS"); // Hello, UnJS!
```
